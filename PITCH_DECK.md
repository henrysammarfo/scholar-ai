## Slide 4: Architecture
**Flow:**
1. **User Request** -> Analyzed by Root Agent.
2. **IQ Checks** -> Confirms crypto definitions via IQ.wiki.
3. **Parallel Tools** -> Tavily executes 3 distinct searches (Overview, Details, Trends).
4. **Writer Agent** -> Synthesizes all tool outputs into a report.
5. **Delivery** -> Streamed to the Anti-Gravity UI.
